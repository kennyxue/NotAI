# AI写作助手 (AI Writing Assistant)

一个基于macOS平台的智能写作助手应用，集成了强大的AI功能，帮助用户提高写作效率和质量。

## 功能特点

- 三栏式布局设计
- 智能AI写作辅助
- 支持富文本编辑
- 实时文件管理
- Markdown支持
- 数学公式编辑
- 表格创建和编辑

## 系统要求

- macOS 12.0 或更高版本
- 最低8GB内存推荐
- 网络连接（用于AI功能）

## 开发环境设置

1. 克隆项目
```bash
git clone [repository-url]
cd AIWritingAssistant
```

2. 安装依赖
- 使用Xcode 14.0或更高版本
- 确保已安装最新版本的macOS开发工具

3. 构建和运行
- 使用Xcode打开`AIWritingAssistant.xcodeproj`
- 选择目标设备为macOS
- 点击运行按钮或按下`Cmd + R`
